Defining Variables of Different Data Types

Here are some examples of javascript literal variable declarations of different data types:

var length = 16;                               // Number assigned by a number literal
var points = x * 10;                           // Number assigned by an expression literal
var lastName = "Johnson";                      // String assigned by a string literal
var cars = ["Saab", "Volvo", "BMW"];           // Array assigned by an array literal
var person = {firstName:John, lastName:Doe};   // Object assigned by an object literal
Function Declarations

Here are some different ways to declare functions:


function superFizzBuzz (array) {
	return array;
};

var superFizzBuzz = function(array) {
	return array;
};

var superFizzBuzz = new Function('array', 'return array')  // No one uses this
For Loops

Here are some exmples of for loops- the first type of for loop is non object specific; you can create a variable to iterate over (a specific amount of times). The For..In loop, however, will loop through properties in an object:


Regular For Loop

for (statement 1; statement 2; statement 3) {
    code block to be executed;
}

for (i = 0; i < 5; i++) {
    text += "The number is " + i + "
";
}

For..In Loop

for (var property_name in some_object) {
   //statements using some_object[property_name];
}

var person = {fname:"John", lname:"Doe", age:25}; 

var text = "";
var x;
for (x in person) {
    text += person[x];
}
While / Do While Loops

Another type of loop are While, and Do While loops, which are the same, except whether it checks the condition at the beginning or the end:

While Loop:

while (condition) {
    code block to be executed
}

while (i < 10) {
    text += "The number is " + i;
    i++;
}

Do While Loop:

do {
    code block to be executed
}
while (condition);

do {
    text += "The number is " + i;
    i++;
}
while (i < 10);
If/Else Statements

Here's how to do an if/else/else if statement, followed by an example

if (condition1) {
    block of code to be executed if condition1 is true
} else if (condition2) {
    block of code to be executed if the condition1 is false and condition2 is true
} else {
    block of code to be executed if the condition1 is false and condition2 is false
}

if (time < 10) {
    greeting = "Good morning";
} else if (time<20) {
    greeting = "Good day";
} else {
    greeting = "Good evening";
}
Defining Objects

Here are some different ways to define Object objects:

// Object literal:
var objectA = {};

var objectB = {index1: 'value 1', index2: 'value 2'};

var myStructure = {
  name: {
    first: "Mel",

  },

// Constructor:
var anObject = new Object();

// Adding/accessing properties:
